# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

### Added

- Initial public extract of **termorg** (Terminal Organiser).
- Kitty provider: list, focus, launch, ambient tab color/title.
- CLI: list/filter, watch, panel, groups, priority, queue, hooks, launch, hints.
- Agent hook ingestion for needs-you / working attention.
- Desktop notifications on rising-edge needs-you.
- Path→group learning and suggestions.

### Known limitations

- Kitty-only provider (multi-provider planned).
- OS-window raise often fails on GNOME Wayland; tab focus still works.
